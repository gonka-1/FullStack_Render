# Proyecto Full Stack - 10 Microservicios

## 1. Microservicio: Profesor
Listar todos: GET https://profesor-service.onrender.com/api/v1/profesores (o /profesor)

Buscar por ID: GET https://profesor-service.onrender.com/api/v1/profesores/{id}

Crear: POST https://profesor-service.onrender.com/api/v1/profesores

Actualizar: PUT https://profesor-service.onrender.com/api/v1/profesores/{id}

Eliminar: DELETE https://profesor-service.onrender.com/api/v1/profesores/{id}
